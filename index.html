Inserindo links

Vamos criar um link para que o nosso usuário conheça o SENAI. O comando para criarmos um link é a TAG a. Se definirmos a target como: _blank: abre a página em uma nova janela/aba; _self: abre a página na mesma janela; _parent: abre a página na mesma janela do link;

Tabelas

A tabela é uma estrutura composta de linhas e colunas, com inúmeras possibilidades de formatação. Para construir uma tabela usamos a TAG table, para construir uma linha usamos a TAG tr, e para criar uma coluna usamos a TAG td.
